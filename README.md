#  AI Student Evaluation System

An AI-powered web application that helps teachers evaluate student assignments automatically using Google Gemini AI. The platform generates scores, detailed feedback, strengths, weaknesses, and improvement suggestions, making the evaluation process faster and more efficient.

---

## 🌐 Live Demo
https://ai-student-evaluation-system.netlify.app/

---

## ✨ Key Features

### 👨‍🏫 Teacher
- Secure Registration & Login
- Create, Edit & Delete Assignments
- View Student Submissions
- AI-based Assignment Evaluation
- Automatic Score Generation
- Detailed Question-wise Feedback

### 👨‍🎓 Student
- Secure Registration & Login
- View Available Assignments
- Submit Assignment Answers
- View AI Evaluation Results
- Track Evaluation History

---

## 🤖 AI Evaluation

The application uses **Google Gemini AI** to evaluate student answers and generate:

- 📊 Score (Out of 10)
- 💬 Detailed Feedback
- ✅ Strengths
- ⚠️ Weaknesses
- 💡 Suggestions for Improvement
